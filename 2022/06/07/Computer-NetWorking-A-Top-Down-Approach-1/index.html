Computer NetWorking:A Top-Down Approach (1)

Chapter 1 Computer Networks and the Internet

1.1 What is the Internet?

In this section,we will discuss what is the Internet? and what is a protocol?

  1. A Nuts-And-Bolts Description

The Internet is a computer network that interconnects billion of computing devices throughout the world. In Internet jargon,these devices are called hosts or end system.

End system are connected by communication links and packet switches.

  • Communication links are made up of different types of physical media. The transmission rate of a link measured in bits/second.
  • When one end system have date to another end system,the sending system segments the data and add header bytes to each segment.
  • A packet switch takes packets arriving on the incoming communication links and forward packets on outgoing communication links.
  • The two most prominent types of packet switches is router and link-layer switched.
    • Link-layer Swiches are used in access networking(LAN).
    • Router are used in the network core.

End system,packet switches and other pieces of the Internet run protocols that control the sending and receiving of information within the Internet.

  1. A Services Description
    The Internet is an infrastructure that provides services to applications.
    End systems attached to the Internet provide a socket interface that speci- fies how a program running on one end system asks the Internet infrastructure to deliver data to a specific destination program running on another end system. This Internet socket interface is a set of rules that the sending program must follow so that the Internet can deliver the data to the destination program.

  2. What Is a Protocol?
    All activity in the Internet that involves two or more communicating remote entities is governed by a protocol.
    “A protocol defines the format and the order of messages exchanged between two or more communicating entities, as well as the actions taken on the transmission and/or receipt of a message or other event.”
